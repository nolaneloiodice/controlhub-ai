# ControlHub AI

ControlHub AI est mon assistant personnel en cours de développement.

L’objectif du projet est de créer progressivement un centre de contrôle personnel basé sur l’IA pour m’aider à apprendre les systèmes, les réseaux, la cybersécurité, Python, Git/GitHub, et à construire un portfolio professionnel.

## Objectifs du projet

- Suivre ma progression en compétences IT
- Générer des tâches d’apprentissage quotidiennes
- Documenter mon évolution
- Créer des projets GitHub valorisables
- Préparer mon profil professionnel
- Évoluer vers un assistant personnel plus complet

## Fonctionnalités actuelles

- Création d’un profil utilisateur
- Ajout de compétences
- Affichage des compétences
- Génération d’une tâche du jour
- Sauvegarde des données en JSON
- Validation du niveau de compétence entre 1 et 5
- Journal d’apprentissage dans `docs/learning-log.md`
