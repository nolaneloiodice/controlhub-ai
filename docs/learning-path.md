# Learning Path

## Contexte

ControlHub AI est né d’un besoin personnel : centraliser ma progression en systèmes, réseaux, cybersécurité et développement Python.

Avant de lancer ce projet, j’ai commencé à explorer plusieurs domaines techniques de manière pratique :

- virtualisation avec VirtualBox
- installation et manipulation d’une machine virtuelle Ubuntu
- commandes Linux de base
- tentative de mise en place d’un serveur Apache
- labs réseaux avec Cisco Packet Tracer
- documentation de projets sur GitHub
- début d’apprentissage cybersécurité avec TryHackMe

## Objectif

L’objectif est d’apprendre en construisant un outil réel.

ControlHub AI me permet de :

- suivre mes compétences
- organiser mes projets
- documenter mes apprentissages
- transformer mes erreurs en progression
- construire un portfolio professionnel cohérent
- préparer une évolution vers les systèmes, réseaux et la cybersécurité

## Parcours actuel

### 1. Systèmes et Linux

J’ai installé Ubuntu dans VirtualBox afin de pratiquer les bases de Linux dans un environnement isolé.

Compétences travaillées :

- virtualisation
- installation d’un système Linux
- navigation dans le terminal
- commandes Linux de base
- installation de paquets
- découverte des services système

Un premier objectif a été d’installer Apache et d’essayer de lancer un serveur web local. Cette étape a révélé des points à approfondir autour des services, des ports, du pare-feu et du diagnostic.

### 2. Réseaux

J’ai ensuite travaillé sur des labs Cisco Packet Tracer pour pratiquer les bases du réseau.

Compétences travaillées :

- adressage IP
- segmentation réseau
- VLAN
- DHCP
- routage
- routage inter-VLAN
- NAT/PAT
- documentation technique sur GitHub

### 3. Cybersécurité

J’ai commencé à explorer TryHackMe afin d’aborder la cybersécurité de manière progressive et pratique.

Objectifs :

- renforcer mes bases Linux et réseau
- comprendre les environnements de lab
- progresser vers la cybersécurité défensive
- apprendre à documenter mes démarches

### 4. ControlHub AI

ControlHub AI est devenu le projet central pour relier ces apprentissages.

Le projet sert à suivre :

- mes compétences
- mes objectifs
- mes projets
- mes progrès
- mes prochaines étapes
